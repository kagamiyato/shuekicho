# 収益帳（配信者向け／アフィリエイター向け）

配信媒体（YouTube・Twitch・17LIVE等）やASP（A8.net・もしもアフィリエイト等）から
ダウンロードした報酬明細CSVを取り込み、確定申告用に集計するツールです。

## 特徴

- ブラウザだけで完結し、サーバーへの通信は一切発生しません（CSVはこのページの中だけで処理されます）
- 静的ファイル1枚（`index.html`）のみで動作し、ビルド不要です

## ローカルで試す

このフォルダの `index.html` をブラウザで直接開くだけで動作します。

## 免責事項

本ツールは確定申告書ではなく、集計を補助するツールです。
所得区分の判定や実際の申告内容については、国税庁の情報をご確認いただくか、
税理士等の専門家にご相談ください。
